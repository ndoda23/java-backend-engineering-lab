# Java Spring Boot Backend Lab 

Welcome to my Backend Showcase repository. This project contains a collection of production-ready, highly testable, and secure Spring Boot REST APIs. It focuses on real-world enterprise challenges like database optimization, custom security filters, and robust business logic architecture.

## 🛠️ Enterprise Tech Stack
* **Framework:** Spring Boot (Spring Web, Spring Security, Spring Data JPA).
* **Database & ORM:** PostgreSQL, Hibernate.
* **Security:** Stateless JWT authentication via HttpOnly Secure Cookies.
